# simple-interest-calculator
A simple web-based application that calculates simple interest using the formula SI = (P × R × T) / 100. Users enter principal, rate, and time to get instant results. Built with HTML, CSS, and JavaScript, it is lightweight, easy to use, and runs directly in a browser.

## Description
This project calculates simple interest using the formula:

Simple Interest = (Principal × Rate × Time) / 100

## Features
- Takes user input
- Calculates simple interest
- Displays result

## How to Run
1. Install Python
2. Run the script:
   python simple_interest.py

## Example
Input:
Principal = 1000  
Rate = 5  
Time = 2  

Output:
Simple Interest = 100
